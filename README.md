# element-finder

You can run with docker using this command:
```
docker compose up -build 
```

Or you can run with npm

```
npm install
npm run build
npm run start:dev
``` 

## Routes

To know about what is your scrap possibilities :)
```
/help
```

The scrap route
```
/scrap?url=<URL>&element=<ELEMENT>
```
